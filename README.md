# Sales Performance Dashboard (Excel)
## Project Overview

This project analyzes the sales performance of executives across multiple regions using Microsoft Excel.
The dashboard provides insights into regional performance, employee sales performance, target achievement, and daily sales trends.

The goal of this project is to transform raw sales data into meaningful business insights using Excel analytics tools such as Pivot Tables, charts, and KPIs.

This project demonstrates my data analysis, data visualization, and business insight generation skills, which are important for a Data Analyst role.

# Objectives

The main objectives of this analysis were:

Analyze sales performance across different regions

Measure target achievement of sales executives

Identify top and low performing employees

Understand daily sales trends

Generate insights that can help improve sales strategy

## Questions (KPIs)

-Who are the top sales executives based on total sales?
-Who are the lowest performing sales executives based on total sales?
-Which sales executives achieved the highest target hit percentage?
-Which sales executives have the lowest target achievement percentage?
-Who are the top 5 sales executives based on target achievement?
-Who are the bottom 5 sales executives based on target achievement?
-hat is the overall sales performance compared to the target?
-How does employee performance vary across the dataset?

# Sales Performance Dashboard (Excel)
## Project Overview

This project analyzes the sales performance of executives across multiple regions using Microsoft Excel.
The dashboard provides insights into regional performance, employee sales performance, target achievement, and daily sales trends.

The goal of this project is to transform raw sales data into meaningful business insights using Excel analytics tools such as Pivot Tables, charts, and KPIs.

This project demonstrates my data analysis, data visualization, and business insight generation skills, which are important for a Data Analyst role.
